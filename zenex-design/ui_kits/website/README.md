# Flowdesks Website UI Kit

A high-fidelity recreation of the public marketing site for flowdesks.ai. Includes:

- **Nav** — sticky top bar with blurred bg, primary CTA
- **Hero** — headline (with Newsreader italic accent), subhead, dual CTAs, orchestration illustration
- **SolutionsGrid** — 7 solution cards using branded category icons
- **StatsStrip** — dark inverse band with three tabular Mono stats
- **HowItWorks** — three-step horizontal flow
- **ContactForm** — work email, company, team size, message; validates inline
- **Footer** — three-column with lockup, link groups, fine print

Open `index.html` to see them composed into a full one-page marketing site.

These are visual recreations only — form submissions, button clicks, and nav links are not wired to a backend.
